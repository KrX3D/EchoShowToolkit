# EchoShowToolkit

PowerShell toolkit for Amazon Echo Show devices running LineageOS.

## Supported Devices

- Echo Show 5 Gen 1 (Checkers)
- Echo Show 5 Gen 2 (Cronos)
- Echo Show 8 Gen 1 (Crown)

## Requirements

- Windows
- PowerShell 5+
- Android Platform Tools (adb.exe)

Place adb.exe either:
- in the toolkit folder
- or in your PATH

## Features

- ADB USB connection
- ADB WiFi connection
- Device information
- Storage check
- Cleanup old APK/BIN files
- APK installation
- F-Droid installation
- ViewAssist installation helper
- LineageOS tweaks
- Home Assistant tweaks
- Screen/Doze tweaks

Run:

.\EchoShowToolkit.ps1
